# ネガポジ判定

## 概要
テキストボックスに文章を入力し判定ボタンを押すと、その文章がポジティブなのかネガティブなのかをスコアとともに出力する。

## 開発環境
Python 3.7.4

## 使用サービス
[単語感情極性対応表](http://www.lr.pi.titech.ac.jp/~takamura/pubs/pn_ja.dic)
